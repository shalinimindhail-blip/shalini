# 🎁 Interactive Birthday Website

A beautiful, romantic, and interactive birthday website designed as a personalized digital gift for your loved one.

## Features

✨ **9 Interactive Pages:**
1. **Welcome Page** - Elegant introduction with floating hearts
2. **Gift Acceptance** - Playful gift acceptance with elusive "No" button
3. **Loading Screen** - Animated progress with love messages
4. **Memory Scrapbook** - Polaroid-style flip cards with memories
5. **10 Reasons** - Reasons why you love them (flip cards)
6. **Our Song** - Embedded music player
7. **Hidden Hearts** - Find all 5 hearts game
8. **Love Letter** - Elegant envelope with love letter
9. **Grand Finale** - Gift unwrapping with celebration animations

## Customization

Edit all content in `data.js`:
- Recipient name
- All text on every page
- Memory photos and captions
- All 10 reasons
- Music link (YouTube or Spotify)
- Secret messages
- Final birthday message
- Colors and styling

## Getting Started

1. Clone this repository
2. Open `index.html` in a web browser
3. Edit `data.js` to personalize content
4. Add your own photos to `assets/images/`

## File Structure

```
.
├── index.html          # Main HTML file
├── style.css          # All styling
├── script.js          # All functionality
├── data.js            # Editable configuration
├── assets/
│   ├── images/        # Add your photos here
│   └── music/         # Optional music files
└── README.md
```
